# JavaScript Calculator (Mini Project)

## Description
This is a beginner-friendly JavaScript calculator created to practice:
- Functions
- Parameters and return values
- Control flow (switch-case)
- Error handling
- Math operations

## Features
- Addition
- Subtraction
- Multiplication
- Division with zero handling
- Square
- Square root with validation

## Technologies Used
- JavaScript (ES6)

## How to Run
1. Clone the repository
2. Open the file in a browser console or Node.js
3. Call the `calculate()` function with required arguments

This project is part of my JavaScript learning journey as a 1st-year CSE student.
